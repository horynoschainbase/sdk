## 🏗️ Architecture

Horyn OS is structured into multiple core layers:

```
/core        → Kernel & system logic
/network     → Communication protocols
/security    → Encryption & validation
/runtime     → Execution environment
/modules     → Extendable components
```

Each layer is designed to work independently while maintaining seamless interoperability.
